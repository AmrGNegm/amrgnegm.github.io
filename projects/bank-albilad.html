Bank Albilad — Enterprise Data Platform

Architecting the data platform that consolidates banking data sources into a governed, single-source-of-truth data warehouse.

The Challenge

Bank Albilad needed a unified data platform that could integrate data from core banking systems, CRM platforms, and operational databases into a single, governed data warehouse. The existing reporting infrastructure relied on manual Excel workflows, with data fragmented across multiple systems and no standardized data model. Business users couldn’t get consistent answers to the same questions.

My Role

Data Integration & BI Architect — responsible for designing and delivering the end-to-end data platform, from source system integration through to self-service Power BI dashboards.

Architecture

  • Star Schema Data Warehouse: Designed a dimensional model (fact and dimension tables) optimized for banking analytics — branch performance, product analytics, and regulatory reporting.
  • Azure Data Factory Pipelines: Built and orchestrated multi-source ingestion pipelines connecting core banking, CRM, and operational systems with proper dependency chains and error handling.
  • Data Governance & SSOT: Established data quality rules, lineage tracking, and a governance framework ensuring all dashboards drew from the same validated sources.
  • Power BI Dashboards: Delivered end-to-end dashboards for branch performance, product analytics, and management reporting — with row-level security and incremental refresh.

Results

  • Consolidated 5+ data sources into a single governed data warehouse
  • Eliminated conflicting numbers between departmental reports through SSOT architecture
  • Reduced manual reporting effort by replacing Excel workflows with automated Power BI dashboards
  • Enabled self-service analytics for business users with governed, validated data

Tech Stack

  • Azure Data Factory
  • Azure SQL Database
  • SQL Server (Star Schema DWH)
  • Power BI (Desktop & Service)
  • DAX & Power Query (M)
  • Row-Level Security
  • Incremental Refresh
